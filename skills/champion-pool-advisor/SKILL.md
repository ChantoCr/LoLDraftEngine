# champion-pool-advisor

## Purpose
Design features related to personal champion pools and recommendation filtering.

## Responsibilities
- define champion pool profile types
- support pool tiers such as main, comfort, playable, emergency
- compare best-overall versus within-pool recommendations
- expose tradeoffs between theoretical best pick and realistic best pick

## Rules
- pool logic must remain optional
- do not conflate player comfort with team-fit value
- clearly explain when a pool-limited pick is weaker or stronger than the absolute best pick
